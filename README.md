# 🎧 Spotify Top 50 World Analytics Dashboard

An interactive **Power BI dashboard** that analyzes global music trends using the Top 50 Spotify songs dataset. This project focuses on transforming raw music data into meaningful insights through data visualization and storytelling.

---

## 🚀 Project Overview

This project explores the **Top 50 songs worldwide** and provides insights into:
- Song popularity trends
- Artist dominance
- Impact of song duration
- Explicit vs non-explicit content performance
- Album type comparison (Singles vs Albums)
- Release year trends

The dashboard is designed with a **modern UI inspired by Spotify**, making it visually appealing and easy to explore.

---

## 📊 Key Features

- 🎤 Top Artists Analysis  
- 🔥 Popularity Distribution  
- ⏱️ Duration vs Popularity Insights  
- 📅 Release Year Trend Analysis  
- 🔞 Explicit vs Non-Explicit Comparison  
- 💿 Album Type Performance  
- 🎯 KPI Cards for quick insights  

---

## 🛠️ Tools & Technologies

- **Power BI** – Dashboard creation and visualization  
- **DAX (Data Analysis Expressions)** – Measures and KPIs  
- **Data Cleaning & Transformation**

---

## 📁 Dataset

The dataset contains the Top 50 Spotify songs with attributes like:
- Date  
- Position  
- Song Name  
- Artist  
- Popularity  
- Duration (ms)  
- Album Type  
- Total Tracks  
- Release Date  
- Explicit Content Indicator  

📌 Dataset Source:  
https://www.kaggle.com/datasets/dhruvildave/spotify-charts
